# 婚約指輪プロジェクト - リサーチ結果

## 1. 素材について

### プラチナ（Pt900 / Pt950）

- **特徴**: 白銀色の美しい輝き。変色に非常に強く、ダイヤモンドの輝きを最も引き立てる
- **純度**: Pt900は90%、Pt950は95%がプラチナ（K18ゴールドの75%より高純度）
- **希少性**: 3gの指輪を作るのに約1トンの原石が必要
- **人気度**: 婚約指輪の約70%がプラチナを選択
- **価格帯**: ゴールドよりやや高め

### ゴールド（K18）

- **イエローゴールド（YG）**: 華やかで温かみのある色合い。カジュアルなファッションにも合う
- **ピンクゴールド（PG）**: 肌馴染みが良く、可愛らしい印象。女性人気が高い
- **ホワイトゴールド（WG）**: プラチナに近い見た目。プラチナより軽く、やや安価
- **強度**: K18はプラチナより硬く、日常使いに向いている

### 素材選びのポイント

- 彼女の普段のアクセサリーの色味（シルバー系 or ゴールド系）に合わせると◎
- 結婚指輪との重ね付けも考慮

## 2. ダイヤモンドの4C

ダイヤモンドは「4C」という国際基準で品質が評価されます。

### カラット（Carat）= 大きさ・重さ

- 1カラット = 0.2g
- **婚約指輪の平均: 0.2〜0.3カラット**
- 0.3ct: 直径約4.3mm（最も人気のサイズ）
- 0.5ct: 直径約5.2mm（やや大きめで存在感あり）
- 予算30〜35万円なら0.2〜0.3ctが現実的

### カット（Cut）= 輝き

- Excellent → Very Good → Good → Fair → Poor の5段階
- **カットが最も輝きに影響するため、最優先で選ぶべき**
- 「Excellent」以上を推奨（予算内で十分選べる）

### カラー（Color）= 色味

- D（完全無色）〜 Z（黄色味あり）の23段階
- **D〜Fが「無色」、G〜Jが「ほぼ無色」**
- G〜Hグレードなら肉眼では差がわからず、コスパが良い

### クラリティ（Clarity）= 透明度

- FL（完全無傷）〜 I3（肉眼で見える内包物）の11段階
- **VS1〜VS2なら肉眼で内包物は見えず、コスパが良い**
- SI1でも十分な透明度

### 4Cの優先順位（おすすめ）

1. **カット** → 輝きに直結。Excellent推奨
2. **カラット** → 見た目の大きさ。0.25〜0.3ct推奨
3. **カラー** → G〜Hで十分
4. **クラリティ** → VS2〜SI1で十分

## 3. デザインの種類

### ソリティア（一粒ダイヤ）

- 婚約指輪の約55%が選ぶ、最も定番のデザイン
- 一粒のダイヤモンドが主役。シンプルで上品
- 立て爪タイプと、爪が低いタイプがある
- どんなファッションにも合わせやすい

### メレ（サイドストーン付き）

- センターダイヤの両サイドに小さなダイヤをあしらったデザイン
- ソリティアより華やかな印象
- アーム部分のメレダイヤが全体の輝きをアップ

### パヴェ

- リングのアーム部分に小さなダイヤを敷き詰めたデザイン
- ゴージャスでキラキラした印象
- 日常使いしやすいものも多い

### エタニティ

- リングの全周（フル）または半周（ハーフ）にダイヤを配置
- 「永遠の愛」を象徴するデザイン
- 結婚指輪としても人気

### ヘイローセッティング

- センターダイヤの周囲を小さなダイヤで囲んだデザイン
- センターダイヤがより大きく見える効果がある
- ゴージャスな印象

### スリーストーン

- 3つのダイヤが並んだデザイン
- 「過去・現在・未来」を象徴
- 華やかで存在感がある

## 4. 人気ブランド詳細

### 国内ブライダル専門店

#### 銀座ダイヤモンドシライシ

- 日本初のブライダルジュエリー専門店（1994年創業）
- 18,000ピースのダイヤ + 150種以上のデザイン
- セミオーダーで数万通りの組み合わせ
- 価格: 21万円台〜
- 全国60店舗以上
- アフターサービス充実

#### アイプリモ（I-PRIMO）

- 成約実績83万組以上
- 120種類以上のデザイン
- プロポーズ後にデザイン変更可能な「パーフェクトプロポーズリング」あり
- 価格: 25万〜35万円が平均

#### エクセルコ ダイヤモンド

- ラウンドブリリアントカットを生み出したブランド
- ダイヤモンドのカット品質に定評
- 価格: 19万円〜
- ハイジュエラーの半額で最高品質が手に入る

#### NIWAKA（俄）

- 京都発の日本ブランド
- 日本の美意識を活かした洗練されたデザイン
- サイズ直し・クリーニングなどアフターサービス充実

#### 4℃（ヨンドシー）

- 手の届きやすい価格帯
- 品質保証・無償アフターサービス
- 普段使いしやすいデザインが豊富

#### ケイウノ（K.UNO）

- 自社工房でデザインから制作まで一貫対応
- フルオーダーが得意（世界にひとつだけ）
- ディズニーコラボデザインも人気

### 海外ハイブランド

#### ティファニー（Tiffany & Co.）

- 世界5大ジュエラーのひとつ
- 「ティファニーセッティング」が有名
- 0.25ct プラチナ: 約30万円
- ブランド認知度が非常に高い

#### カルティエ（Cartier）

- 「王の宝石商」として知られる
- 30万円台からエントリーモデルあり
- 予算35万円ではやや選択肢が限られる

## 5. おすすめの候補（予算30〜35万円）

### コスパ重視なら

1. **銀座ダイヤモンドシライシ** — 0.3ct + 好みのデザインが選べる。選択肢の豊富さが魅力
2. **エクセルコ ダイヤモンド** — カット品質が最高。同予算でダイヤの質が高い
3. **4℃** — 0.3ct以上も狙える。普段使いしやすいデザイン

### ブランド力重視なら

1. **ティファニー** — 0.25ct セッティングリングが約30万円。誰もが知るブランド
2. **カルティエ** — エントリーモデルが30万円台。予算上限に近い

### デザイン・こだわり重視なら

1. **NIWAKA** — 和の美意識が光る独自デザイン
2. **ケイウノ** — 世界にひとつのオーダーメイド（納期に注意）
3. **アイプリモ** — デザイン変更可能なプロポーズリングが便利
