# 婚約指輪プロジェクト - 計画・スケジュール管理

## プロジェクト概要

- プロポーズ予定日: **2026年5月30日（土）**
- 目的: 婚約指輪を選び、プロポーズ時にプレゼント
- 予算: 30万円〜35万円（オプション込み）

---

## スケジュール（逆算）

プロポーズ日から逆算し、余裕を持ったスケジュールを組みます。
既製品なら2週間〜1ヶ月、セミオーダーなら3週間〜1.5ヶ月、フルオーダーなら3〜6ヶ月の納期が一般的です。

| フェーズ                      | 期間         | やること                                       | 備考                                       |
| ----------------------------- | ------------ | ---------------------------------------------- | ------------------------------------------ |
| **Phase 1: リサーチ**         | 3/13         | ブランド・デザイン・素材の調査、候補リスト作成 | 本ドキュメント群で完了                     |
| **Phase 2: 彼女への提案**     | 3/13         | プレゼン資料を彼女に送付                       | カジュアルに共有                           |
| **Phase 3: 店舗訪問（下見）** | 3/14         | 候補ブランドの店舗に来店、試着・雰囲気確認     | まずは見てみる程度。購入決定はしない       |
| **Phase 4: 候補絞り込み**     | 3月下旬〜4月 | 下見の感想をもとにふたりで候補を絞る           | 必要に応じて再訪問                         |
| **Phase 5: 購入・オーダー**   | 4月中旬      | 指輪を決定し注文                               | 既製品orセミオーダーなら5/30に十分間に合う |
| **Phase 6: 受け取り**         | 5月中旬      | 完成品を受け取り、刻印・サイズ確認             | プロポーズの1〜2週間前が目安               |
| **Phase 7: プロポーズ**       | 5/30         | 指輪を渡してプロポーズ！                       |                                            |

## 注意事項

- **納期リスク**: フルオーダーの場合は3〜6ヶ月かかるため、5/30には間に合わない可能性あり。既製品またはセミオーダーを推奨。
- **サイズ確認**: 彼女の指のサイズが不明な場合、来店時にリングゲージで計測してもらう or プロポーズ後にサイズ調整する方法もある。
- **来店予約**: 人気店は土日は混雑するため、事前予約がおすすめ。
