# 婚約指輪プロジェクト - 予算管理

## 予算概要

| 項目                              | 金額               |
| --------------------------------- | ------------------ |
| **総予算**                        | **30万円〜35万円** |
| 指輪本体（リング + ダイヤモンド） | 25万〜33万円       |
| オプション（刻印・サイズ調整等）  | 0〜2万円           |

---

## 予算配分の考え方

婚約指輪の価格は「リング枠（素材・デザイン）」+「ダイヤモンド」で構成されます。

### リング枠の目安

- プラチナ（Pt900/Pt950）: 5万〜10万円
- K18ゴールド（YG/PG/WG）: 4万〜8万円

### ダイヤモンドの目安（0.2〜0.3ct の場合）

- 国内ブライダル専門店: 15万〜25万円
- 海外ハイブランド: 20万〜30万円

---

## オプション費用

| オプション       | 費用目安   | 備考                                 |
| ---------------- | ---------- | ------------------------------------ |
| **サイズ調整**   | 0〜1万円   | 多くのブランドで購入後初回無料       |
| **刻印**         | 0〜5,000円 | 購入時の刻印は多くのブランドで無料   |
| **クリーニング** | 0円        | ほとんどのブランドで永久無料         |
| **歪み直し**     | 0〜5,000円 | アフターサービスに含まれることが多い |
| **石の留め直し** | 0〜1万円   | ブランドにより保証期間が異なる       |

---

## コストを抑えるポイント

1. **ブライダル専門店を選ぶ**: 海外ハイブランドよりダイヤモンドの品質に対してコスパが良い
2. **カラット優先で4Cを選ぶ**: 見た目に影響が大きいカラットとカットを優先し、カラー・クラリティは中間グレードでも十分
3. **アフターサービスを確認**: サイズ調整や刻印が無料のブランドを選べばオプション費用を節約できる
4. **来店時にフェアや特典を確認**: ブランドによっては来店特典や成約特典がある
